# CanvasFlow | Professional Obsidian Canvas Editor

CanvasFlow is a lightweight, web-based viewer and editor for [Obsidian .canvas files](https://obsidian.md/canvas). It provides a smooth user experience for visualizing and organizing your notes, ideas, and workflows in a canvas-style interface.

## ‚ú® Features

- **Import .canvas files**: Simply drag and drop your `.canvas` files to visualize them instantly.
- **Interactive Viewing**: Pan and zoom around your canvas with ease.
- **Dual Modes**:
  - **Viewer Mode**: Clean interface for exploring your canvas.
  - **Editor Mode**: Modify node positions and connections.
- **Hand-drawn Style**: Toggle between a professional look and a organic, hand-drawn aesthetic (using Rough.js).
- **Markdown Support**: Nodes render Markdown content with full formatting.
- **Sanitized Output**: Integrated DOMPurify to ensure safe rendering of node content.
- **Fast & Responsive**: Built with pure JavaScript, CSS, and HTML for maximum performance.

### Prerequisites

You only need a modern web browser to run CanvasFlow.

3. Drag and drop a `.canvas` file onto the interface to begin.
